# PrintScreen-Saver
### Швидко зберігай свої скріншоти не запускаючи Paint!
![alt text](https://cloud.githubusercontent.com/assets/24297249/26431349/9d2a855a-40fc-11e7-9c1e-b56329a6401b.png "я помираю. допоможіть")

___

### How to:
+ Натисни на клавіатурі кнопку Print Screen (prt sc). 
+ Натисни на "велика сіра область з чорною рамкою".
+ Обери папку для збереження та назву файла.
+ Натисни кнопку "Зберегти".
